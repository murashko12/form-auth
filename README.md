# Task Manager

Простоая admin-панель

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/murashko12/form-auth.git
```

## Запуск backend
1. Перейдите в папку backend

```bash
cd backend
```

2. Установить все необходимые зависимости

```bash
npm install
```

3. Запуск

```bash
npm run start
```

## Запуск frontend
1. Перейдите в папку frontend

```bash
cd frontend
```

2. Установить все необходимые зависимости

```bash
npm install
```

3. Запуск

```bash
npm run dev
```
